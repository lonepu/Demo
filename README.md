# Flutter Quiz App 🚀

This is a simple Quiz Application built with Flutter. Users can answer questions and receive a score based on their answers. After the quiz, the total score is displayed, and users have the option to restart the quiz.

## Features ✨

* **Multiple Choice Questions:** Each question has several answer options. ❓

* **Score Tracking:** Calculates scores based on correct answers. 💯

* **Result Summary:** Shows total score and a message at the end of the quiz. 📊

* **Restart Option:** Allows users to restart the quiz from the beginning. 🔄

* **Android Only:** Currently developed for the Android platform. 🤖

## Technologies Used 💻

* **Flutter:** Mobile application development framework. 💙

* **Dart:** Programming language for Flutter. 🎯

## Getting Started 🏁

To run this project on your local machine, follow these steps:

### Prerequisites ✅

* [Flutter SDK](https://flutter.dev/docs/get-started/install) (Stable channel)

* [Android Studio](https://developer.android.com/studio) or [VS Code](https://code.visualstudio.com/) (with Flutter plugin)

* Android Emulator or Android Device (for debugging) 📱

### Installation ⬇️

1. Clone this repository:

```

git clone [https://github.com/lonepu/Demo.git]
cd Demo

```

2. Install dependencies:

```

flutter pub get

```

### Running the App ▶️

1. Start an Android Emulator or connect your Android Device.

2. Run the app:

```

flutter run

```

Alternatively, you can run it via your IDE (Android Studio/VS Code).
